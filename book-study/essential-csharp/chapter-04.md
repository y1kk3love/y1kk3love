# 4장 — 연산자와 흐름 제어

## 연산자 종류

| 연산자 타입 | 연산자 |
|-------------|--------|
| 단항 | `+`, `-` |
| 산술 | `+`, `-`, `*`, `/`, `%` |
| 할당 | `=`, `+=`, `-=`, `*=`, `/=`, `%=` |
| 증감 | `++`, `--` |
| 논리 | `&&`, `\|\|`, `!` |
| 관계/비교 | `<`, `>`, `==`, `!=`, `>=`, `<=` |
| 비트 | `&` (AND), `\|` (OR), `^` (XOR) |
| Shift | `>>`, `<<` |
| 조건 | `?:`, `??` |

---

## 연산자 우선순위 (높은 순서)

| 연산자 | 연산 유형 | 결합 |
|--------|-----------|------|
| `a[x]` `f(x)` `x.y` `new` `++` `--` (후위) | 식 | 왼쪽 → 오른쪽 |
| `(T)x` `+` `-` `~` `!` `++` `--` (전위) | 단항 | 오른쪽 → 왼쪽 |
| 형식 캐스팅 | 단항 | 오른쪽 → 왼쪽 |
| `*` `/` `%` | 곱하기 | 왼쪽 → 오른쪽 |
| `+` `-` | 더하기 | 왼쪽 → 오른쪽 |
| `<<` `>>` | 비트 시프트 | 왼쪽 → 오른쪽 |
| `<` `>` `<=` `>=` `is` `as` | 관계 | 왼쪽 → 오른쪽 |
| `==` `!=` | 같음 | 왼쪽 → 오른쪽 |
| `&` | 비트 AND | 왼쪽 → 오른쪽 |
| `^` | 비트 제외 OR | 왼쪽 → 오른쪽 |
| `\|` | 비트 포함 OR | 왼쪽 → 오른쪽 |
| `&&` | 논리 AND | 왼쪽 → 오른쪽 |
| `\|\|` | 논리 OR | 왼쪽 → 오른쪽 |
| `??` | Null 병합 | 왼쪽 → 오른쪽 |
| `? :` | 조건식 | 오른쪽 → 왼쪽 |
| `=` `*=` `/=` `%=` `+=` `-=` 등 | 단순 및 복합 할당 | 오른쪽 → 왼쪽 |
| `,` | 순차적 계산 | 왼쪽 → 오른쪽 |
