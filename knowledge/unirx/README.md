# UniRX

UniRX란, C#용 비동기 처리를 위한 라이브러리를 의미합니다.  
[Observer](https://ko.wikipedia.org/wiki/%EC%98%B5%EC%84%9C%EB%B2%84_%ED%8C%A8%ED%84%B4) 패턴을 기반으로 설계되어 있습니다.  
시간에 관련된 처리 및 실행 타이밍이 중요한 곳에서 처리를 쉽게 작성할 수 있도록 되어 있습니다.
